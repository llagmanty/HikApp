# HikApp
Privacy-First Hiking Companion A lightweight iOS application focused on absolute user privacy, providing essential trail tracking and performance statistics without compromise. Design Philosophy From a UX standpoint, the interface is deliberately minimalist, prioritizing context while eliminating data entry friction. The entire experience begins and ends with the hike itself - no login, no setup, no persistent digital footprint. Technical Highlights • Native iOS implementation using Swift and UIKit/SwiftUI. • Seamless map integration powered by Apple Maps API for real-time location visualization. • Secure, localized session handling: all tracking data exists exclusively in-memory during active sessions. • Ephemeral data model: session statistics are displayed for immediate review and subsequently destroyed, leaving zero data at rest.
